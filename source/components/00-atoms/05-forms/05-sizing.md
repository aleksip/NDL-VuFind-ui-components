#### Description

A styled input text element which can be used to achieve different sized inputs. Input can be a required field.

#### Properties
| Prop          | Type       | Required |
| ------------- | ---------- | -------- |
| inputName     | `String`   | Yes      |
| labelClass    | `String`   | No       |
| label         | `String`   | Yes      |
| inputClass    | `String`   | No       |
| placeholder   | `String`   | Yes      |
| required      | `Boolean`  | No       |
| type          | `String`   | Yes      |
