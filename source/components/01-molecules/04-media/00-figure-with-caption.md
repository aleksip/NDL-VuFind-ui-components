#### Description

An image with caption text. Caption can be aligned to either left or right.

#### Properties
| Prop          | Type       | Required |
| ------------- | ---------- | -------- |
| rounded       | `Boolean`  | No       |
| imgSrc        | `String`   | Yes      |
| imgAlt        | `String`   | No       |
| height        | `String`   | No       |
| width         | `String`   | No       |
| align         | `String`   | No       |
| caption       | `String`   | Yes      |
