#### Description

A dropdown element, which is particularly used for menus and filters. Can be set to outlined styling with `outline` property.

#### Properties

| Prop    | Type       | Required |
| ------- | ---------- | -------- |
| outline | `Boolean`  | No       |
| label   | `String`   | Yes      |
| items   | `Object[]` | Yes      |
| submit  | `String`   | Yes      |
