#### Description

A dropdown element, which is particularly used for menus and filters. Can be set to outlined styling with `outline` property.

#### Properties

| Prop          | Type       | Required |
| ------------- | ---------- | -------- |
| outline       | `Boolean`  | No       |
| dropdownClass | `String`   | No       |
| labelClass    | `String`   | No       |
| label         | `String`   | Yes      |
| iconLabel     | `String`   | No       |
| items         | `Object[]` | Yes      |
| fallbackLabel | `String`   | Yes      |
